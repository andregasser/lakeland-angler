# P1-Musterset

## Bestand

- `Icons/`: vier SVG-Master mit PNG-Prüfexports
- `Fische/`: freigestelltes Black-Crappie-Muster
- `Montagen/`: Bobber-Rig-SVG mit PNG-Prüfexport

## Provenienz

Die Icons und die Montage wurden als deterministische SVG-Dateien nach `Design/16_P1_Musterset_v1.0.md` erstellt.

Die Fischillustration wurde im eingebauten Imagegen-Modus erzeugt. Der ausgewählte zweite Proof verwendete folgende Kernaussage: anatomisch plausibler adulter Black Crappie in vollständiger linker Seitenansicht, unregelmäßige dunkle Flecken statt vertikaler Bänder, natürliche silbrig-olivfarbene Färbung, weiches Tageslicht, keine Umgebung oder Beschriftung und vollständig deckende Flossen vor einem einfarbig blauen Chroma-Key-Hintergrund. Der Hintergrund wurde anschließend lokal entfernt; das finale PNG besitzt Transparenz und wurde auf 3600 × 2400 Pixel skaliert.
