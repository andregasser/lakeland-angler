# Redaktionelle Vorlagen

Diese Vorlagen setzen die Bucharchitektur aus `Projekt/05_Bucharchitektur_v1.0.md` um.

## Verwendung

1. Passende Vorlage in das vorgesehene Kapitelverzeichnis unter `Inhalte/` kopieren.
2. Datei nach dem projektspezifischen Benennungsschema benennen.
3. YAML-Metadaten ausfüllen.
4. Platzhalter in spitzen Klammern ersetzen.
5. Nicht benötigte optionale Felder entfernen.
6. Quellen vollständig in den Produktionsdaten dokumentieren.

Vorlagen sind Produktionshilfen und werden nicht direkt in das Buchlayout übernommen.
